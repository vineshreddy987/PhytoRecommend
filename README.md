# 🌿 PhytoRecommend — AI Phytoremediation Plant Recommender

An AI-powered system that recommends the best plant species for remediating
contaminated soil. Given site conditions (pollutant element, BCF, soil pH,
organic matter, exposure duration), the model predicts the top-5 plant species
most suitable for phytoremediation.

Trained on **36,599 real phytoremediation studies** across **56 plant species**
and **15 contaminant elements** (Pb, Cd, Zn, Cu, As, Ni, Cr, Co, Hg, Se, Mn,
Fe, Tl, Mo, Sb).

## Quickstart

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Train the models
```bash
# Step 1 — create the train/test split
python pipeline/00_setup_and_split.py \
    --data data/phytoremediation_dataset.csv \
    --label species \
    --output_dir models/split_data

# Step 2 — train per-element models (recommended, unbiased)
python pipeline/06_retrain_per_element.py

# Optional — run full benchmark (XGBoost + ANN + TabTransformer + Transformer)
python pipeline/01_train_xgboost.py
python pipeline/02_train_ann.py
python pipeline/03_train_tabtransformer.py
python pipeline/04_train_self_attention_transformer.py
python pipeline/05_aggregate_results.py

# Optional — run full class-balancing report
python pipeline/07_full_pipeline_report.py
```

### 3. Start the web app
```bash
python webapp/app.py
```
Then open **http://127.0.0.1:5000** in your browser.

## Inputs

| Field | Description | Example |
|---|---|---|
| Contaminant Element | Pollutant in the soil | `Pb` (Lead) |
| Soil Concentration | Amount in soil (mg/kg) | `250` |
| Plant Concentration | Amount absorbed by plant (mg/kg) | `45` |
| BCF | Bioconcentration Factor = plant/soil | `0.18` (auto-calculated) |
| Soil pH | pH of contaminated soil | `5.2` |
| Organic Matter | % organic matter in soil | `1.5` |
| Exposure Duration | Days plant grows in soil | `60` |

**BCF > 1** = hyperaccumulator (plant takes up more than soil has) — ideal for phytoremediation.

## Output

Top-5 recommended plant species with confidence scores, ranked by suitability
for the given contamination conditions.

## Model Architecture

The system uses **15 per-element XGBoost classifiers** — one per contaminant.
Training each model on element-specific data eliminates cross-element species
dominance bias (the "always predict Sunflower" problem).

### Features used
- `log(soil_concentration)`, `log(plant_concentration)`, `log(BCF)`
- `soil_ph`, `organic_matter_pct`, `log(duration_days)`
- `pH × log(BCF)` interaction
- `log(concentration) × log(BCF)` interaction
- `BCF category` (low / moderate / hyperaccumulator)

### Class imbalance handling
- Rare species (< 50 records) removed
- Inverse-frequency **sample weights** applied during training
- **SMOTE oversampling** for minority classes

### Benchmark results (global model)

| Model | Accuracy | Top-3 | ROC-AUC |
|---|---|---|---|
| **XGBoost** | **49.5%** | **69.3%** | **0.948** |
| TabTransformer | 38.9% | 57.8% | 0.915 |
| ANN (MLP) | 24.2% | 43.5% | 0.818 |
| 12-Head SAT | 19.9% | 42.0% | 0.720 |

## Notes

- Model `.pkl` files are **gitignored** (too large for Git, ~160 MB total).
  Run `pipeline/06_retrain_per_element.py` to regenerate them.
- The `data/phytoremediation_dataset.csv` is the cleaned version of the
  merged phytoremediation database.
- All random seeds are fixed at `42` for reproducibility.
